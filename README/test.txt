1. Инициализация данных: Мы создаём массив lines, который содержит имя файла в первой строке и его содержимое в последующих строках.

2. Вывод имени файла: Программа выводит имя файла, которое хранится в первой строке массива lines.

3. Объединение содержимого: 

   • Мы создаём массив content, в который собираем все строки, начиная со второй.

   • После каждой строки добавляем символ переноса строки (\n), чтобы сохранить форматирование.

   • В конце добавляем символ конца строки (0), чтобы корректно завершить строку.

4. Вывод содержимого: Программа выводит объединённое содержимое файла.

5. Проверка наличия слова "Привет": 

   • Используется стандартная функция strstr для поиска подстроки в строке. Если слово найдено, выводится соответствующее сообщение.
